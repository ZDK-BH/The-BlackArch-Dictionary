# bluetooth

**Tools in this category:** 26

**Category use:** Bluetooth security auditing, device inventory, protocol study, and lab testing of your own devices.

## Tools

- [blue-hydra](./blue-hydra.md) — A Bluetooth device discovery service built on top of the bluez library.
- [bluebugger](./bluebugger.md) — An implementation of the bluebug technique which was discovered by Martin Herfurt.
- [bluediving](./bluediving.md) — A Bluetooth penetration testing suite.
- [bluefog](./bluefog.md) — A tool that can generate an essentially unlimited number of phantom Bluetooth devices.
- [bluelog](./bluelog.md) — A Bluetooth scanner and sniffer written to do a single task, log devices that are in discoverable mode.
- [bluepot](./bluepot.md) — A Bluetooth Honeypot written in Java, it runs on Linux.
- [blueprint](./blueprint.md) — A perl tool to identify Bluetooth devices.
- [bluescan](./bluescan.md) — A Bluetooth Device Scanner.
- [bluesnarfer](./bluesnarfer.md) — A bluetooth attacking tool.
- [bluphish](./bluphish.md) — Bluetooth device and service discovery tool that can be used for security assessment and penetration testing.
- [braces](./braces.md) — A Bluetooth Tracking Utility.
- [bss](./bss.md) — Bluetooth stack smasher / fuzzer.
- [bt_audit](./bt_audit.md) — Bluetooth audit
- [btcrack](./btcrack.md) — The world's first Bluetooth Pass phrase (PIN) bruteforce tool. Bruteforces the Passkey and the Link key from captured Pairing exchanges.
- [btlejack](./btlejack.md) — Bluetooth Low Energy Swiss-army knife.
- [btproxy-mitm](./btproxy-mitm.md) — Man in the Middle analysis tool for Bluetooth.
- [btscanner](./btscanner.md) — Bluetooth device scanner.
- [carwhisperer](./carwhisperer.md) — Sensibilise manufacturers of carkits and other Bluetooth appliances without display and keyboard for the possible security threat evolving from the use of standard passkeys.
- [ghettotooth](./ghettotooth.md) — Ghettodriving for bluetooth.
- [hidattack](./hidattack.md) — HID Attack (attacking HID host implementations).
- [obexstress](./obexstress.md) — Script for testing remote OBEX service for some potential vulnerabilities.
- [redfang](./redfang.md) — Finds non-discoverable Bluetooth devices by brute-forcing the last six bytes of the devices' Bluetooth addresses and calling read_remote_name().
- [sparrow-wifi](./sparrow-wifi.md) — Next-Gen GUI-based WiFi and Bluetooth Analyzer.
- [spooftooph](./spooftooph.md) — Designed to automate spoofing or cloning Bluetooth device Name, Class, and Address. Cloning this information effectively allows Bluetooth device to hide in plain sight.
- [tbear](./tbear.md) — Transient Bluetooth Environment Auditor includes an ncurses-based Bluetooth scanner (a bit similar to kismet), a Bluetooth DoS tool, and a Bluetooth hidden device locator.
- [ubertooth](./ubertooth.md) — A 2.4 GHz wireless development board suitable for Bluetooth experimentation. Open source hardware and software. Tools only.
