# debugger

**Tools in this category:** 10

**Category use:** Debugging local binaries, crash triage, exploit-development education, and malware-analysis labs.

## Tools

- [edb](./edb.md) — A cross platform AArch32/x86/x86 debugger.
- [electric-fence](./electric-fence.md) — A malloc(3) debugger that uses virtual memory hardware to detect illegal memory accesses.
- [gdbgui](./gdbgui.md) — Browser-based gdb frontend using Flask and JavaScript to visually debug C, C++, Go, or Rust.
- [heaptrace](./heaptrace.md) — Helps visualize heap operations for pwn and debugging.
- [ollydbg](./ollydbg.md) — A 32-bit assembler-level analysing debugger.
- [rr](./rr.md) — A Record and Replay Framework.
- [saleae-logic](./saleae-logic.md) — Debug happy.
- [shellnoob](./shellnoob.md) — A toolkit that eases the writing and debugging of shellcode.
- [vivisect](./vivisect.md) — A Python based static analysis and reverse engineering framework.
- [voltron](./voltron.md) — UI for GDB, LLDB and Vivisect's VDB.
