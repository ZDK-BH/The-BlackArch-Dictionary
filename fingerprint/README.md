# fingerprint

**Tools in this category:** 30

**Category use:** Identifying software, versions, frameworks, services, CMSs, and exposed technology stacks.

## Tools

- [asp-audit](./asp-audit.md) — An ASP fingerprinting tool and vulnerability scanner.
- [blindelephant](./blindelephant.md) — A web application fingerprinter. Attempts to discover the version of a (known) web application by comparing static files at known locations
- [cms-explorer](./cms-explorer.md) — Designed to reveal the specific modules, plugins, components and themes that various cms driven websites are running.
- [detectem](./detectem.md) — Detect software and its version on websites.
- [dhcpf](./dhcpf.md) — Passive DHCP fingerprinting implementation.
- [dnsmap](./dnsmap.md) — Passive DNS network mapper.
- [fl0p](./fl0p.md) — A passive L7 flow fingerprinter that examines TCP/UDP/ICMP packet sequences, can peek into cryptographic tunnels, can tell human beings and robots apart, and performs a couple of other infosec-related tricks.
- [fpdns](./fpdns.md) — Program that remotely determines DNS server versions.
- [ftpmap](./ftpmap.md) — Scans remote FTP servers to identify what software and what versions they are running.
- [htrosbif](./htrosbif.md) — Active HTTP server fingerprinting and recon tool.
- [httprint](./httprint.md) — A web server fingerprinting tool.
- [kolkata](./kolkata.md) — A web application fingerprinting engine written in Perl that combines cryptography with IDS evasion.
- [lbmap](./lbmap.md) — Proof of concept scripts for advanced web application fingerprinting, presented at OWASP AppSecAsia 2012.
- [mercury](./mercury.md) — Network metadata capture and analysis.
- [mwebfp](./mwebfp.md) — Mass Web Fingerprinter.
- [neighbor-cache-fingerprinter](./neighbor-cache-fingerprinter.md) — An ARP based Operating System version scanner.
- [nerva](./nerva.md) — Fast service fingerprinting CLI for 170+ protocols (TCP/UDP/SCTP).
- [nimbostratus](./nimbostratus.md) — Tools for fingerprintinging and exploiting Amazon cloud infrastructures.
- [ntp-fingerprint](./ntp-fingerprint.md) — An active fingerprinting utility specifically designed to identify the OS the NTP server is running on.
- [operative](./operative.md) — Framework based on fingerprint action, this tool is used for get information on a website or a enterprise target with multiple modules (Viadeo search,Linkedin search, Reverse email whois, Reverse ip whois, SQL file forensics ...).
- [scannerl](./scannerl.md) — The modular distributed fingerprinting engine.
- [sinfp](./sinfp.md) — A full operating system stack fingerprinting suite.
- [smtpmap](./smtpmap.md) — Tool to identify the running smtp software on a given host.
- [smtpscan](./smtpscan.md) — An SMTP scanner
- [spartan](./spartan.md) — Frontpage and Sharepoint fingerprinting and attack tool.
- [thcrut](./thcrut.md) — Network discovery and OS Fingerprinting tool.
- [tls-fingerprinting](./tls-fingerprinting.md) — Tool and scripts to perform TLS Fingerprinting.
- [tls-prober](./tls-prober.md) — A tool to fingerprint SSL/TLS servers.
- [xprobe2](./xprobe2.md) — An active OS fingerprinting tool.
- [zgrab2](./zgrab2.md) — Fast Application Layer Scanner.
