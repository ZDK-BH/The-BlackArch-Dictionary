# tunnel

**Tools in this category:** 18

**Category use:** Authorized network tunneling, lab connectivity, debugging, and controlled remote access tests.

## Tools

- [chisel](./chisel.md) — A fast TCP tunnel over HTTP.
- [chownat](./chownat.md) — Allows two peers behind two separate NATs with no port forwarding and no DMZ setup on their routers to directly communicate with each other
- [ctunnel](./ctunnel.md) — Tunnel and/or proxy TCP or UDP connections via a cryptographic tunnel.
- [dns2tcp](./dns2tcp.md) — A tool for relaying TCP connections over DNS.
- [fraud-bridge](./fraud-bridge.md) — ICMP and DNS tunneling via IPv4 and IPv6.
- [icmptx](./icmptx.md) — IP over ICMP tunnel.
- [ip-https-tools](./ip-https-tools.md) — Tools for the IP over HTTPS (IP-HTTPS) Tunneling Protocol.
- [ligolo-ng](./ligolo-ng.md) — An advanced, yet simple, tunneling tool that uses a TUN interface.
- [matahari](./matahari.md) — A reverse HTTP shell to execute commands on remote machines behind firewalls.
- [morxtunel](./morxtunel.md) — Network Tunneling using TUN/TAP interfaces over TCP tool.
- [multitun](./multitun.md) — Tunnel arbitrary traffic through an innocuous WebSocket.
- [neo-regeorg](./neo-regeorg.md) — Improved version of reGeorg, HTTP tunneling pivot tool
- [ngrok](./ngrok.md) — A tunneling, reverse proxy for developing and understanding networked, HTTP services.
- [oniongrok](./oniongrok.md) — Onion addresses for anything.
- [regeorg](./regeorg.md) — The successor to reDuh, pwn a bastion webserver and create SOCKS proxies through the DMZ. Pivot and pwn.
- [stegosip](./stegosip.md) — TCP tunnel over RTP/SIP.
- [vstt](./vstt.md) — VSTT is a multi-protocol tunneling tool. It accepts input by TCP stream sockets and FIFOs, and can send data via TCP, POP3, and ICMP tunneling.
- [xfltreat](./xfltreat.md) — Tunnelling framework.
